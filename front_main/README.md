## Install

---

- create React App

```bash
npx create-react-app my-app
cd my-app
npm start
```

- React-Router-Dom

```bash
npm i react-router-dom
```

- styled-components

```bash
npm install --save styled-components
```

- axios

```bash
npm install axios
```

- Redux

```bash
npm install @reduxjs/toolkit
```
